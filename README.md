<h1>Note Taker</h1>

<h3>Description</h3>
-A node application using express where you can add, view, and delete notes. This application uses express routing with GET, POST, and DELETE methods.

<h3>Technologies Used</h3>
-HTML<br>
-CSS<br>
-Bootstrap CSS Framework<br>
-JavaScript<br>
-Node.js<br>
-Express<br>
